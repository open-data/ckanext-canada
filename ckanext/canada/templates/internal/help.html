Help

This page provides instructions on using the Open Data Metadata Registry to add dataset metadata records to the Government of Canada Open Data Portal.

Additional information on creating metadata for the Registry is found in the Government of Canada Open Data Portal Metadata Implementation Guide, 2nd Edition.

Adding a Dataset

To add a dataset to the Open Data Portal, you must create a metadata record for that dataset and add it to the Registry. To add a metadata record:

  1. Click 'Add Dataset'
  2. Ensure that the dataset you are adding meets the requirements in the Dataset Criteria list. This list is also found under Guidelines in the Registry.
  3. Follow the instructions provided in the Government of Canada Open Data Portal Metadata Implementation Guide.

Reviewing your Datasets

To review your dataset metadata records or to see the complete list of records, click Dashboard. The records are listed in the order in which they were entered in the Registry.

Reviewing the Datasets of your Department or Agency

To see the complete set of metadata records belonging to your department or agency, click Organizations.

To see a record within the Registry, click the name of the dataset.

Searching the Registry

Use Search to find metadata records published in the Registry. This feature is the same as Advanced Search on the Open Data Portal website.

To search the Registry:

  1. Type your search keywords under Search open data .

The list of results shows datasets matching your search query in the order in which they were entered in the Registry (chronological).

Browsing the Registry

Use the Browse feature to find metadata records published in the Registry. This feature is similar to Browse on the Open Data Portal website.

You can browse the Registry in three ways:

  1. By department. Click on a department or agency name to see all datasets published by that organization.
  2. By subject. Click on a subject to find all datasets 'tagged' with that subject.
  3. By datatype. Click on a datatype name to find all datasets in that format.

The list of results displays datasets in the order in which they were entered in the Registry (chronological).

Aide

Le présent document fournit des instructions sur l'utilisation du Registre de métadonnées pour le Portail de données ouvertes du gouvernement du Canada.

Pour de plus amples renseignements, consultez le document intitulé Guide de mise en place des métadonnées pour les ressources Web, 2e édition

Ajouter un jeu de données

L'ajout d'un ensemble de données sur le Portail de données ouvertes comprend l'incorporation au Registre d'un enregistrement de métadonnées relatif à l'ensemble de données. Voici la procédure à suivre :

  1. Cliquez sur Ajouter un jeu de données.
  2. Veillez à ce que l'ensemble de données que vous voulez verser au Portail soit conforme à la liste des critères applicables aux ensembles de données. Cette liste fait également partie des lignes directrices relatives au Registre.
  3. Suivez les instructions fournies dans le document intitulé Guide de mise en place des métadonnées pour les ressources web, 2e édition

Examiner vos ensembles de données

Pour examiner vos ensembles de données et les enregistrements de métadonnées connexes, ou encore pour consulter la liste complète des enregistrements, cliquez sur Tableu de bord. Les enregistrements sont présentés dans l'ordre où ils ont été versés au Registre.

Examiner les ensembles de données de votre ministère ou organisme

Pour examiner l'ensemble des enregistrements de métadonnées de votre ministère ou organisme, cliquez sur Organisations.

Pour consulter un enregistrement dans le Registre, cliquez sur le nom de l'ensemble de données correspondant.

Faire une recherche dans le Registre

Utilisez la fonction Recherche pour trouver des enregistrements de métadonnées publiés dans le Registre. Cette fonction est la même que la fonction Recherche avancée sur le site Web du Portail de données ouvertes.

Pour effectuer une recherche dans le Registre :

  1. Tapez des mots clés dans la fenêtre de recherche de données.

Une liste des ensembles de données correspondant à vos critères de recherche sera affichée; les ensembles de données figureront dans l'ordre où ils ont été versés au Registre.

Naviguer à l'intérieur du Registre

Utilisez la fonction de navigation pour trouver des enregistrements de métadonnées publiés dans le Registre. C'est une fonction similaire à celle que comporte le Portail de données ouvertes.

Vous pouvez naviguer dans le Registre selon trois paramètres :

  1. Par ministère - Cliquez sur le nom d'un ministère ou d'un organisme pour voir quels ensembles de données il a publiés.
  2. Par sujet - Cliquez sur un sujet pour voir tous les ensembles de données auxquels ce sujet a été associé.
  3. Par type de données - Cliquez sur un format de données pour voir tous les ensembles de données produits dans ce format.

Les ensembles de données sont affichés dans l'ordre où ils ont été versés au Registre (ordre chronologique).
